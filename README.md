# NoSQL Datahack Lab

Repo to build virtual machine for NoSQL Module Lab of Datahack BigData Master Class

## Sofware
### Service versions
* elasticsearch: 7.1.5
* kibana: 7.1.5
* mongoDB: 4.2
* neo4j: 3.5.8

## How to run
Access to root project folder and run:
```
$ docker-compose up -d
```

To stop environment run:
```
$ docker-compose down
```
